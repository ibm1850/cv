## Customization

### Update Information

Edit `index.html` to update:
- Personal information
- Work experience
- Skills
- Education
- Contact details

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    --secondary-color: #8b5cf6;
    /* ... other variables */
}
```

### Add Sections

Simply add new sections following the existing structure in `index.html` and style them in `styles.css`.
